# Networking Basics

Understand how data moves across the internet: TCP/IP, HTTP/HTTPS, DNS, and how requests travel from a browser to a server and back. Learn what latency and bandwidth mean practically, not just theoretically. This matters because almost every design decision eventually traces back to network constraints.
