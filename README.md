# Introduction

I started this gitbook to keep track of all the knowledge I acquire throughout my career, this time talking about Systems Design.

### About me

I started this journey at the university, studying Systems Engineer at Bolivian Catholic University "San Pablo". I learnt a lot about Software Development, Networking, Computer Architecture, Operating Systems, and much more, but I became most passionate about Software Development.

After finishing the university, I got my first job as a Software Developer, having as my tech stack:

* Angular for the frontend
* Java + PL/SQL for the backend

Everything was great, but after completing many tasks, I realized that development wasn't all I wanted to do. That's when I decided to keep learning, and systems design was the first big subject I tackled. It was really challenging, and that's why I'm sharing here all my experience for those who are starting this journey.

I hope you find it useful.
